# PWA Icons

This directory contains the PWA (Progressive Web App) icons for TernantApp.

## Quick Setup

### Option 1: Generate from SVG (Recommended)

Run the generation script:
```bash
cd frontend/public/icons
./generate-icons.sh
```

This requires either ImageMagick or Inkscape installed on your system.

### Option 2: Use Online Tool

1. Visit [RealFaviconGenerator](https://realfavicongenerator.net/)
2. Upload `icon.svg`
3. Configure settings:
   - Platform: Web App Manifest
   - Android: Yes
   - iOS: Yes
4. Download the package
5. Extract and place icons in this directory

### Option 3: Manual Installation

Install ImageMagick:
```bash
# Ubuntu/Debian
sudo apt install imagemagick

# macOS
brew install imagemagick

# Then run:
./generate-icons.sh
```

## Required Icon Sizes

The PWA requires the following icon sizes (generated by the script):
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Icon Design Guidelines

- Use the provided `icon.svg` as a template
- Maintain aspect ratio
- Use high contrast colors for visibility
- Consider safe zones for different platforms
- Test on both light and dark backgrounds
